**EduPro Elite** es la solución definitiva de **PATRIC SOFT ®** diseñada para optimizar el flujo de trabajo de los docentes de nivel secundario y técnico. El sistema permite generar evaluaciones profesionales, planificaciones áulicas y rúbricas de calificación en segundos, eliminando la carga administrativa tediosa.

---

## 🚀 Características Principales

* **Generador de Evaluaciones:** Crea consignas de desarrollo y mix de preguntas basadas en ejes temáticos específicos.
* **Sistema Anti-Copia (Variantes):** Capacidad de generar **TEMA A** y **TEMA B** con barajado automático de preguntas.
* **Planificación Áulica:** Redacción automática de objetivos y fundamentación pedagógica.
* **Rúbricas Profesionales:** Tabla de criterios de calificación integrada para una evaluación transparente.
* **Modo Impresión Perfecto:** Diseño optimizado para imprimir en hojas A4 con formato institucional impecable.
* **Control de Suscripción:** Sistema de prueba gratuita limitado a 3 usos para gestión de monetización del software.

---

---

## 📦 Instalación y Uso

1. Descargue el archivo `EduPro_Elite.html`.
2. Ábralo en cualquier navegador moderno (Chrome, Edge o Firefox).
3. Complete los datos de la institución y la materia.
4. Presione **"PROCESAR MATERIAL"**.
5. Imprima directamente o guarde como PDF usando el botón de impresión.

---

## 💰 Modelo de Negocio (PATRIC SOFT)

El software incluye un **Firewall de Pago** basado en el uso local:
* **3 Usos Gratuitos:** Para que el docente compruebe la calidad del material.
* **Bloqueo de Seguridad:** Al alcanzar el límite, el sistema solicita una clave de acceso o contacto directo vía WhatsApp para la activación de la versión Full.
